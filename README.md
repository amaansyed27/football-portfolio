# Divyanshu Mittal — Football Portfolio

A cinematic, Barça-inspired developer portfolio built around a continuous football journey. The Al Rihla ball follows the scroll through player profile, technical formation, experience and projects before the final shot into goal.

## Stack

- React + TypeScript + Vite
- Three.js + React Three Fiber + Drei
- GSAP ScrollTrigger + Lenis
- CSS motion with reduced-motion support

## Local development

```bash
npm install
npm run dev
```

Create a production build with `npm run build`.

The same production build is checked automatically by GitHub Actions on pushes and pull requests.

## Asset note

The supplied GLB assets are used as narrative elements. The World Cup model was geometry-optimised for web delivery while preserving the original source outside the repository.
